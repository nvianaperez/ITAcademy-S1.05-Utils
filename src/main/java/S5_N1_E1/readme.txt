********* COMPILING **********

Open cmd
Type --> cd <package directory> Ex.: cd "C:\_PROGRAMMING\BackEnd_ITAcademy\Sprint1_S5\src\main\java\S5_N1_E1"
Type --> javac *.java --> compile all the java classes
Check that a .class file is created in the same package Ex.: "DirExercise1.class"

********* RUNNING **********
From "package directory" of the project
Type --> java <java class.java> "argument"

Ex.: java DirExercise1.java "C:\_PROGRAMMING\BackEnd_ITAcademy\Sprint1_S5"

Result:
C:\_PROGRAMMING\BackEnd_ITAcademy\Sprint1_S5\src\main\java\S5_N1_E1>java DirExercise1.java "C:\_PROGRAMMING\BackEnd_ITAcademy\Sprint1_S5"
