Open cmd
Type --> cd <package directory> Ex.: cd "C:\_PROGRAMMING\BackEnd_ITAcademy\Sprint1_S5\src\main\java\S5_N1_E3"

Check that a .class file is created in the same package Ex.: "DirExercise3.class"
